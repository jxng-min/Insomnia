# Insomnia(불면증)
</div>

### 개요

수원대학교 '컴퓨터과학특강' 수업의 기말고사 대체 과제를 위한 게임입니다.<br></br> 
Unity 5를 활용하여 2D Top-Down뷰 공포게임을 제작하는 것이 목표입니다.

<br></br>

### 프로젝트 요약

| 정보 | 설명 |
|-----|-------|
| 게임 명 | 불면증(Insomnia) |
| 게임 장르| 2D, 스릴, 공포 |
| 사용 유니티 버전| Unity 5(2022.3.46f1) → Unity 6(6000.0.26f1) |
