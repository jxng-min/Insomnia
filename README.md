# Insomnia(불면증)
</div>

### 개요

수원대학교 '컴퓨터과학특강' 수업의 기말고사 대체 과제를 위한 게임입니다.

Unity 5를 활용하여 2D Top-Down뷰 공포게임을 제작하는 것이 목표입니다.

2025.03.16) 기존 코드를 리팩토링했습니다.

<br></br>
