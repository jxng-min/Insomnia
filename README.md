# Insomnia(불면증)

### 개요

대학교 수업의 과제 활동의 개인 프로젝트 '불면증'입니다. 유니티 엔진 5.0을 활용하여 2D Top-Down 뷰 공포 게임을 제작하는 것이 목표입니다.
